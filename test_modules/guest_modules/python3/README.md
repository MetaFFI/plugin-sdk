# Python3 Guest Test Module

This guest module exercises Python language features and MetaFFI entity coverage.

Contents:
- `module/`: multi-file Python package with broad language feature coverage.
- `single_file_module.py`: single-file module variant to test simple loading.
- `tests/`: unit tests that validate exposed entities and behavior.
- `third_parties.md`: instructions for optional third-party library tests.

Build artifacts:
- `.py` sources and compiled `.pyc` under the build output directory.
