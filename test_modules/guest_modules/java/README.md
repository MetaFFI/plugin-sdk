# Java Guest Test Module

This guest module exercises Java language features and MetaFFI entity coverage.

Contents:
- `src/`: Java sources (multi-file module).
- `tests/`: Java unit test with a runnable `main`.
- `third_parties.md`: optional third-party tests to be invoked by hosts.
- `special_primitives.md`: Java collections/types to be validated by hosts.

Build artifacts:
- Compiled `.class` files and a jar under `test_bin/`.
